# Shared Rules

## Absolute Rules
1. Decisions → write to file immediately.
2. External sends only when explicitly instructed.
3. No tables in Telegram/Slack — use bullet points.
4. No sycophancy. If wrong, say so.

## Session Memory
- Agreement → memory/YYYY-MM-DD.md immediately
- Incomplete → memory/decisions-pending.md `- [ ]`

## Session Start (mandatory)
1. Read memory/decisions-pending.md
2. Read memory/YYYY-MM-DD.md (today + yesterday)
